###  万字文档+学生公寓电费信息小程序+Uniapp+SSM+Vue
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、Vue2、Uniapp、ElementUI、HTML、JavaScript、CSS、图片上传

系统功能模块：3个角色
本系统包含管理员、公寓管理员、学生三个角色，
管理员角色：公寓管理员管理、学生管理、楼层信息管理、用电情况管理、缴费清单管理、电费统计管理、公寓报修管理、系统管理功能。
   - 公寓管理员管理：查看、添加、删除公寓管理员信息，如账号、姓名、性别等。
   - 学生管理：管理学生信息，如账号、姓名、楼层号、联系方式等，支持详情查看与删除操作。
   - 楼层信息管理：维护楼层信息，包括新增、修改、删除楼层记录。
   - 用电情况管理：监控学生用电情况，包括用电量、金额、余额等，支持详情查看与管理。
   - 缴费清单管理：管理学生的缴费记录，如单号、用电量、金额、缴费状态等，可进行详情查看与删除。
   - 电费统计管理：汇总和分析电费数据，提供统计报表。
   - 公寓报修管理：处理公寓维修请求，跟踪报修状态。
   - 系统管理：进行系统设置、用户权限管理等后台操作。
公寓管理员角色：首页查看、用电情况管理、个人中心功能。
   - 首页：查看重要通知及系统概览。
   - 用电情况：查看所管辖公寓的用电情况，协助管理。
   - 个人中心：管理个人账号信息及查看相关通知。
学生角色：首页查看、用电情况查询、个人中心功能。
   - 首页：浏览公告、获取重要信息。
   - 用电情况：查询个人宿舍的用电量和费用详情。
   - 个人中心：管理个人信息，如查看缴费记录、报修记录等。
